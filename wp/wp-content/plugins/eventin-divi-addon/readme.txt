##Notable Features You can get in Divi addons##

Event List – Show multiple events on your page with an event listing feature. You can list related events to let users choose from similar events they want to attend.

**Events Tab** – Create event tabs with an event tab widget.
**Schedule List** – Insert schedule lists to show the event agenda.
**Schedule Tab** – Insert event schedules in tabs format.
**Speaker List** – Show your event speakers to your guests by creating a speaker listing on your event page.
**Advanced Search** – Filter events on the archive page with advanced search.
**Event Calendar** – Calendar brings a better user experience by providing straight designs. Users will be able to see all the available live events and select the specific date to see which events are under that event.

= Should I install Eventin Plugin for the addons? =

YES! The addons work with Eventin. To get full functionality, you will need to install the Eventin plugin.


== Build Process ==

This plugin uses npm-based build tools (webpack) to generate the production JavaScript files located in the `scripts` directory, including:

- scripts/frontend-bundle.min.js
- scripts/builder-bundle.min.js

The human-readable source code for these bundles is included in this plugin, primarily as React/JSX modules under:

- /includes/modules/

These modules are compiled and bundled via the build configuration defined in `package.json`.

Example source:
- Advanced Search module: /includes/modules/AdvancedSearch/AdvancedSearch.jsx
